# stonks_bot
A Discord Bot for Paper Trading stocks using the IEX Finance API.

Created by Phi-Long Bui and Kevin Dai.

# Setup Instructions
To add this bot to your server click on this link:
        https://tinyurl.com/y5ud8q6c
